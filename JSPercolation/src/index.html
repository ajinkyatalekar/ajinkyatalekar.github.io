Percolation Visualizer











How it worksarrow_drop_down

1. It generates a grid of n * n elements. Each site represents a space that can be occupied by water. The system is said to 'percolate' if a site at the top is connected to a site at the bottom.
2. Till the system percolates, a site is opened at random throughout the grid. This is when water is able to flow from top to bottom through the open sites.

How its coded:
I used weighted union-find to store the site data in an array, and connected each site to their neighbor when they are opened.
See the GitHub Repo for more info and a Java version in which you can paint sites to open.
